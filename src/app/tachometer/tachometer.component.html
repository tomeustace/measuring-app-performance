Tachometer Even if you run the same JavaScript, on the same browser, on the same machine, on the same day, you'll still get a different result every time. But if you take enough repeated samples and apply the right statistics, you can reliably identify even tiny differences in runtime. Running Benchmarks

Alter builds (A / B) with varying algorithms/packages etc to test against.

  • ng build --output-path=dist/appA
  • ng build --output-path=dist/appB
  • serve -p 3000, serve -p 3001
  • tachometer --config benchmarks.json
  • load trace logs chrome://tracing
